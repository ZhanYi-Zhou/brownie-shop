# 馡夢小食光 - 純手工布朗尼專賣店 🍫

## ✨ 關於我們

馡夢小食光是一家專注於製作純手工布朗尼的甜點店。我們堅持使用純天然食材，不添加防腐劑，為您帶來最純粹的美味體驗。每一塊布朗尼都由師傅親手製作，嚴格控制品質，確保每一口都能品嚐到最純正的巧克力風味。

## 🍰 產品特色

- 純天然巧克力製作
- 口感濕潤綿密
- 多種口味選擇
- 新鮮現做
- 無添加防腐劑
